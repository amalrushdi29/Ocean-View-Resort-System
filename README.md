# 🌊 Ocean View Resort System

A full-stack Hotel Management Web Application built with Java Servlets, JSP, and MySQL.

## 📋 About The Project
Ocean View Resort System is a web-based hotel management system that handles room bookings, guest management, staff administration, billing, and reservations.

## ✨ Features
- 🔐 Admin Login & Authentication
- 🏨 Room Management (Add, Edit, Update Status)
- 👥 Guest Management (Check-in & Check-out)
- 📅 Reservation & Booking Management
- 💰 Billing & Receipt Generation (PDF)
- 👨‍💼 Staff Management
- 📊 Admin Dashboard with Statistics

## 🛠️ Technologies Used
| Layer | Technology |
|-------|-----------|
| Frontend | JSP, HTML, CSS, JavaScript |
| Backend | Java Servlets (Jakarta EE) |
| Database | MySQL |
| Server | Apache Tomcat v10.1 |
| Testing | JUnit 6 |
| Libraries | iTextPDF, MySQL Connector, JSON |
| IDE | Eclipse |

## 📁 Project Structure
```
OceanViewResort/
├── OceanController/    # Servlet Controllers
├── OceanDAO/           # Database Access Layer
├── OceanModel/         # Bean/Model Classes
├── OceanTesting/       # JUnit Test Classes
├── webapp/
│   ├── css/            # Stylesheets
│   ├── js/             # JavaScript Files
│   └── *.jsp           # JSP View Pages
└── WEB-INF/lib/        # JAR Dependencies
```

## 🗄️ Database
- Database: MySQL
- Tables: Admin, Guest, Room, Reservation, Billing, Staff
